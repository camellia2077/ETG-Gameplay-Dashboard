## [v0.3.5] - 2026-07-05

### 新增功能 (Added)
* 新增玩家靠近掉落的 `拾取物(pickups)` 时在屏幕上显示详细 `物品图鉴(pickup info)` 提示信息的功能，支持通过控制面板的 `常规(general)` 页面进入 `物品图鉴设置(pickup info config)`，对物品品质、类型、效果、协同、摘要、备注等信息分段进行自定义显隐配置。
* 新增 `EnableActiveItemGrantVerboseLogs` 调试配置键与 `docs/operations/logging-active-item-grant.md` 说明文档，用于捕获主动道具发放及槽位扩容链路的诊断日志。
* 新增 `拾取物(pickups)` 页面中搜索框右侧的“清除”(Clear) 按钮，并补充对应的手柄焦点导航支持。

### 技术改进/重构 (Changed/Refactor)
* 重构主动道具发放逻辑，玩家获取 `主动道具(active items)` 前，优先通过向 `player.ownerlessStatModifiers` 添加 `AdditionalItemCapacity` 属性的 `StatModifier` 来修改主动道具容量上限，解决连续获取多个主动道具时导致的道具掉落或容量回滚问题。
* 重构 GUI 滚动视图样式，将控制面板中分散的滚动条背景、滑块等界面样式逻辑提取并统一封装在 `SharedScrollViewStyles.cs` 静态类中，提高视图组件复用性。
* 重构物品图鉴导出器，在原有字段基础上增加 `EnglishDisplayName` 与 `GameDisplayName`，并在导出时额外输出专用于对照名录的 `RandomLoadout.pickup-names.json` 且写入 `gameLanguageCode` 字段。
* 重构语言检测与本地化，新增 `GuiText.GameLanguageCode` 属性，绕过面板 UI 语言覆盖选项以获取当前游戏运行的真实语言代码，确保物品图鉴服务能正确提取对应的本地化资源。
* 重构项目关于、致谢等文档，在 `README.md`、`README_en.md` 和 `THIRD_PARTY_NOTICES.md` 中补充引用数据源 `wiki.gg` 以及 `Lynx3x/etg-itemtips-cn` 的版权致谢声明。

### 修复 (Fixed)
* 修复简中本地化配置文件中，将游戏中原本翻译不一致的 `空包弹` 统一纠正为官方标准的 `空响弹(Blank)`。
