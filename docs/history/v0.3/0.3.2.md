## [v0.3.2] - 2026-06-28

### 新增功能 (Added)
* 新增全开地图功能，开启后可显示关卡内所有房间（包括隐藏房间），并使所有地图传送点立即可用。
* 新增在开局物品（Start Items）中自定义添加掉落物（Loot）的功能，支持配置宝箱钥匙、老鼠钥匙、最大生命值上限、空响弹及护甲等。
* 新增常规吃枪怪（Gunber Muncher）与邪恶吃枪怪（Evil Muncher）的生成支持。
* 新增房间（Room）子分类中的 `Neutral` (中立) 分页，用于专门放置常规吃枪怪与邪恶吃枪怪的生成项。
* 新增设置（Settings）页面中的 `Controller Help / 手柄说明` 次级页面，详细罗列当前版本固定的手柄操作映射规则（包括 `R3` 按下打开控制台、`A` 确认、`B` 返回/关闭、`D-pad` 移动焦点、`LB` 切换分类页、`L3 + R3` 保留原版 Blank 空响弹逻辑）。
* 新增设置（Settings）与关于（About）页面中的项目仓库链接、最新 Release 下载地址以及“本项目开源、免费、无广告，禁止倒卖”的版权免责声明。
* 新增传送（Teleport）面板顶部的操作提示文案，并补全该面板的完整手柄导航支持（`D-pad` 选择楼层、`A` 确认传送、`B` 关闭面板）。

### 技术改进/重构 (Changed/Refactor)
* 重构手柄输入与控制台导航规则：
  * 修改控制台开启逻辑，将开启方式调整为按下 `R3`，并移除了先前的 `Select/Back + Start` 组合按键快捷开启路径。
  * 统一手柄的关闭与返回操作为 `B` 键。
  * 将控制台导航映射调整为 `D-pad` 移动焦点、`A` 键确认、`LB` 键切换分类页。
  * 手柄输入明确仅基于 360 手柄映射进行处理，不再兼容其他非标准手柄映射方案。
  * 移除了 `RB` 键切换分类页的功能，以防与游戏内默认开火操作冲突。
  * 移除了“控制台弹出按键”的可配置选项，将手柄弹出逻辑完全固定。
  * 允许 `L3 + R3` 组合键输入不被拦截，以维持游戏原本的空响弹（Blank）释放逻辑。
* 重构菜单焦点导航输入机制：
  * 手柄导航改为仅认 `D-pad` 移动焦点，键盘导航新增方向键支持。
  * 停用 `WASD` 键以及手柄左摇杆（即 `Horizontal` / `Vertical` 轴）推动焦点的功能，避免玩家在移动角色时误触界面焦点。
* 优化弹药模式（Ammo Mode）位置与交互逻辑：
  * 调整弹药模式在控制面板中的显示位置。
  * 当弹药模式切换至 `NoConsume` (免除子弹消耗) 状态时，自动禁用换弹模式，以防止零子弹消耗无法自然触发换弹带来的潜在 Bug（手动按 `R` 键换弹除外）。
* 优化界面按钮与排版视觉效果：
  * 当战斗（Combat）选项选中切换至非 `Off` 状态时，动态改变按钮颜色以提供更清晰的视觉反馈。
  * 将设置页的作者标签文本由 `Author GitHub / 作者 GitHub` 调整为 `Author / 作者`。
  * 将设置页中作者、项目仓库及 Release 下载等文本的颜色统一修改为纯白色，取消先前的低不透明度设计以提升可读性。
  * 同步将“本项目开源、免费、无广告，禁止倒卖”声明的文本从 JSON 本地化资源文件移出，改为代码硬编码内置。
  * 因页面内容扩充，适当调大了关于页面和设置页面的面板显示高度。
* 重构滚动条 UI 视觉样式：
  * 为物品浏览器与开局物品（Start Items）等滚动区域接入自定义滚动条样式，整体视觉配色与控制面板主色调对齐。
  * 将滚动条拖拽块（Thumb）颜色设定为固定色，避免因悬停（Hover）或拖拽状态改变颜色。
  * 全面接管滚动条轨道、拖拽块及方向指示按钮的样式，避免 Unity 默认 IMGUI 皮肤样式混入。
* 为 `R3` 开启、`D-pad` 导航、`B` 返回等操作补全了详细的调试日志记录，方便后续输入问题的排查与复现。
* 优化房间宝箱生成交互流程（修改涉及 `InGameCommandController.Room.cs`、`InGameCommandController.CommandPage.cs`、`InGameCommandController.State.cs`）：
  * 点击任意品质宝箱按钮（或手柄焦点移至该品质宝箱并按 `A`）直接触发生成，不再需要选择特定类型后再点击额外的“生成宝箱”按钮。
  * 界面上移除右下角多余的宝箱生成按钮以及“当前等级：...”的状态文本显示。
  * Chest 页按钮现在直接调用 `ExecuteSpawnChest(player, logger, chestTier)`。

### 修复 (Fixed)
* 修复 `D-pad` 在控制台内无法进行焦点导航的缺陷，改为优先读取 `BraveInput` 的方向输入而非 Unity 原生输入轴。
* 修复在部分次级菜单中按下 `B` 键无法正常返回上一级菜单的问题。
* 修复传送（Teleport）面板的外层/内层手柄导航焦点切换不一致的异常。

### 弃用/删除 (Deprecated/Removed)
* 删除控制面板底部早期残留的“按回车生成；再次按键盘按键或手柄组合可关闭”提示文本。
* 删除了中英文本地化资源文件中已无用的 `gui.command.hint.submit` 翻译键。
* 移除了已废弃的宝箱生成焦点 ID `cmd.room.spawn_chest`。
* 移除了界面中对宝箱等级选择翻译键 `gui.room.selected_tier` 的引用。
