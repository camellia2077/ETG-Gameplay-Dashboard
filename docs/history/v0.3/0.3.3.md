## [v0.3.3] - 2026-06-29

### 新增功能 (Added)
* 新增 `Settings -> Keyboard Help / 键盘说明` 页面，集中说明控制面板当前的键盘操作规则。
* 新增 `Settings -> Advanced Tools / 高级工具` 页面，用于承载手动命令输入框与 `Grant` 按钮，避免主控制面板中的输入框误抢焦点。
* 新增 `Insert` 作为确认键、`Delete` 作为返回键的键盘导航支持，并为相关输入补充调试日志。
* 新增角色页（`Characters`）、开局物品页（`Loadout Editor`）及其子页面、拾取物浏览页（`Pickups`）的完整键盘焦点导航能力。
* 新增对 `Pickups` 浏览页搜索框、分类筛选、品质筛选、子筛选与结果列表的完整键盘焦点支持。
* 新增拾取物浏览与开局物品相关列表的自动滚动跟随逻辑，保证焦点下移时滚动区域同步跟随。
* 新增 `EnableCommandPanelHealthVerboseLogs` 调试开关与对应运行文档，用于排查控制面板内血量/护甲动画反复播放、输入串入切枪、以及最大血量回滚恢复等问题。

### 技术改进/重构 (Changed/Refactor)
* 将主 `Control Panel` 中的手动命令输入能力迁移到 `Settings -> Advanced Tools / 高级工具`，并移除主面板中对应输入区域。
* 将 `Random` 按钮从命令输入区域迁移到 `General` 分类中，并改名为 `Random Item / 随机物品`。
* 将 `Player Resources / 人物资源` 统一重命名为 `Pickups / 拾取物`，同步更新页面标题、按钮文案、提示文本与相关日志描述。
* 调整 `Settings` 页面结构，新增 `Advanced Tools / 高级工具` 入口，并为会进入详情页的入口补充 `INS / DEL` 提示。
* 重构键盘方向键导航逻辑，支持长按重复触发，使长列表可持续下移并自动滚动到列表末尾。

### 修复 (Fixed)
* 修复 `+1 Key` 在部分情况下误判玩家已拥有无限钥匙的问题，移除无限钥匙检查与相关提示，仅保留实际加钥匙功能。
* 修复新建开局物品预设时未正确避开内置默认 `Preset / 预设` 的命名冲突问题；新增预设现在会从 `Preset-1 / 预设-1` 开始尝试命名，并在代码中补充说明注释。
* 修复角色页在使用键盘选择进入后，方向键、`Insert` 与 `Delete` 无法工作的缺陷。
* 修复开局物品主页面、详情页、随机池详情页、开局拾取物详情页在使用键盘操作时方向键、`Insert` 与 `Delete` 无法工作的缺陷。
* 修复从 `Loadout Editor` 进入 `Pickups` 物品添加浏览页后，方向键、`Insert` 与 `Delete` 失效的问题。
* 修复 `Pickups` 浏览页焦点继续下移时列表不自动滚动的问题。
* 修复控制面板打开后菜单导航输入会串入 ETG 本体切枪输入的问题；该串入会让运行时最大血量短暂回滚，再被 override 逻辑恢复，从而导致 HUD 反复播出“加心 / 加甲”动画。控制面板现在会在显示期间施加临时输入 override 来隔离菜单导航。

### 弃用/删除 (Deprecated/Removed)
* 删除主 `Control Panel` 页面中“手动命令输入已移动到 Advanced Tools”提示文本及其对应本地化键。
* 删除主控制面板中早期残留的手动命令输入框位置，不再在首页直接接受命令文本输入。
* 删除 `WASD`、手柄左摇杆与手柄右摇杆驱动控制面板焦点选择的逻辑，仅保留方向键与 `D-pad` 作为菜单导航输入。
* 删除命令输入框使用 `Enter` 直接提交生成的快捷方式，统一改为使用 `Grant` 按钮或 `Insert` 确认当前焦点。
