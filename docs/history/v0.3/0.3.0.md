## [v0.3.0] - 2026-06-20

### 新增功能 (Added)
* 新增有装弹动画与无装弹动画的自动换弹功能，允许玩家在射击完毕时自动进行换弹操作。
* 新增在 F7 命令控制面板中实时展示玩家当前角色各项属性（Character Stats）的功能。
* 新增辅助调节开关，支持开启或关闭枪械图鉴（Ammonomicon）的开启与关闭动画。
* 新增游戏调试辅助开关，加入无敌模式与不消耗子弹选项。
* 新增开局自定义物品配置界面，支持直接在游戏内通过图形化列表选取初始装备，并支持创建、命名与保存自定义初始装备预设（Presets）。
* 新增主动道具（Active）冷却方式的筛选分类，支持按 Any / Uses / Damage / Time / Room 方式进行精细过滤（对应 `ActiveNumberOfUses`、`ActiveDamageCooldown`、`ActiveTimeCooldown` 与 `ActiveRoomCooldown` 字段）。

### 技术改进/重构 (Changed/Refactor)
* 优化 F7 命令控制面板的 UI 整体布局，调整页面元素间距与结构，增强可读性与操作流畅度。
* 调整品质等级（Quality）按钮的默认排序逻辑，将其排列顺序调整为 `S -> A -> B -> C -> D -> Any Q -> Special -> Excluded`，使其更符合玩家直觉。
